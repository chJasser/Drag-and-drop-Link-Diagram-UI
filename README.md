## Server Setup

### Local Machine

1. Navigate to the `server` directory:

    ```bash
    cd server
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the server in development mode:

    ```bash
    npm run start:dev
    ```

### Containers

Start the server using Docker Compose:

```bash
docker-compose up
 ```
